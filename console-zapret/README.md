# Альтернатива для Discord, так называемый "Zapret"

1. [Разархивируем](https://github.com/vic-kk/instr/raw/refs/heads/main/console-zapret/DisUnblock.zip) (или копируем папку из архива) в любое место
2. Запускаем файл `discord`.
3. Соглашаемся со всеми вопросами системы (если они будут)
4. Открытое окно игнорируем или сворачиваем  
5. Готово

> [!TIP]
> В архиве лежит иконка ДИСа.  
> - Можно создать новый ярлык для файла (discord) из пункта 2
> - Заменить у ярлыка картинку на иконку ДИСа
> - Перенести этот ярлык, куда вам будет удобно и запускаться уже с ярлыка
